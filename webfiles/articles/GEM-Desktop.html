{ "author" : "Marcio Teixeira", "license" : "http://creativecommons.org/licenses/by-nc-sa/4.0/", "emulators" : ["pce-ibmpc-xt"] } = GEM Desktop = GEM/1 was released for the IBM PC by Digital Research Inc. (DRI) in 1985. GEM is not a complete operating system, but rather a graphical interface that runs on top of DOS. The interface used in GEM was so similar to the Macintosh that Apple successfully sued for copyright infringement[1]. As a result of this lawsuit, DRI removed visual effects and overlapping windows from subsequent releases. { "icons" : [ ["GEM/1 Desktop", "boot-hd", "/disks/ibmpc-gem-1.2.pimg", {"emulator-args" : {"-c" : "pce-config-5160-gem-1.2.cfg"} }] ] } === See also: === * [[Atari 1040ST|Atari-1040ST.html]] - GEM was ported to the 68000 microprocessor by Atari and was included with Atari's ST line of computers in 1985. == About the Emulator == { "icons" : [ ["PCE Home Page", "hyperlink", "http://www.hampa.ch/pce"] ], "class" : "float-right" } The IBM PC emulator is a JavaScript port of Hampa Hug's PCE (PC Emulator)[2].
