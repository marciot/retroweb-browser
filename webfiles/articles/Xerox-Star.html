= About the Xerox Star 8010 = { "icons" : [ ["DigiBarn Xerox Star Page", "hyperlink", "http://www.digibarn.com/collections/systems/xerox-8010"] ] } The Xerox Star, built by Xerox Corporation in 1981, was the grand daddy of the modern graphical user interface that heavily influenced the Apple Macintosh and every graphical interface after it. The photo is by Dave Curbow, downloaded from DigiBarn[1] == Where are the demo disks? == Sadly, I don't have an emulator for this computer, so there are no demo disks.
