About VisiCalc

VisiCalc, originally released for the Apple II in 1979, was an early killer app that brought the full potential of the personal computer to businesses. You can explore VisiCalc inside the simulated IBM PC.

VisiCalc Tutorial

  1. Type “A:” to tell DOS to access the floppy disk drive
  2. Type “RUN” to read the license agreement and start up VisiCalc
  3. Review the VisiCalc Reference Cards to get started
  4. Type “/SQY” to quit VisiCalc
